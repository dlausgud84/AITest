| 항목       | 값                         |
|------------|----------------------------|
| 접속 URL   | http://localhost:3000      |
| 권장 브라우저 | Chrome 최신 버전          |

---

## 2. 로그인 / 로그아웃

### 로그인

1. 브라우저에서 접속 URL을 입력합니다.
2. 로그인 화면(`/login`)에서 아이디와 비밀번호를 입력합니다.
3. **로그인** 버튼을 클릭하면 대시보드로 이동합니다.

### 로그아웃

- 우측 상단 사용자 아이콘 클릭 → **로그아웃** 선택
- 로그아웃 후 로그인 페이지로 자동 이동됩니다.

---

## 3. 화면 구성

```
+--------------------------------------------------+
| 상단 헤더 (로고, 사용자 정보)                     |
+----------+---------------------------------------+
| 좌측     |  콘텐츠 영역                           |
| 사이드바 |  (각 메뉴 화면)                        |
| (메뉴)   |                                        |
+----------+---------------------------------------+
```

- **사이드바**: 주요 메뉴 네비게이션
- **콘텐츠 영역**: 선택한 메뉴의 화면 표시

---

## 4. 메뉴별 사용법

### 대시보드 (`/`)

- 시스템 현황 요약 정보를 표시합니다.

### 메뉴 관리 (`/menu-management`)

- 시스템 메뉴를 등록/수정/삭제합니다.
- 테이블에서 행을 선택하면 상세 정보가 표시됩니다.
- **등록**: 상단 **등록** 버튼 클릭 → 입력 폼 작성 → **저장**
- **수정**: 테이블 행 우측 **수정** 버튼 클릭
- **삭제**: 테이블 행 우측 **삭제** 버튼 클릭 → 확인 팝업에서 **확인**

### 작업지시 (`/workorder`)

- 생산 작업지시를 조회하고 관리합니다.

### 생산 (`/production`)

- 생산 실적을 입력하고 조회합니다.

### 품질 (`/quality`)

- 품질 검사 결과를 입력하고 조회합니다.

### 설비 (`/equipment`)

- 설비 현황 및 가동 이력을 조회합니다.

### 기준정보 (`/master`)

- 품목, 공정, 설비 등 기준 데이터를 관리합니다.

### 시스템 설정 (`/settings`)

- 시스템 환경 설정을 관리합니다.

### 사용자 관리 (`/user-management`)

- 사용자 계정을 등록/수정/삭제합니다.

### 키오스크 (`/kiosk`)

- 현장 작업자용 키오스크 화면입니다.

---

## 5. 세션 타임아웃 안내

- 일정 시간 동안 활동이 없으면 자동으로 로그아웃됩니다.
- 타임아웃 발생 전 경고 팝업이 표시되며, **연장** 버튼을 클릭하면 세션이 유지됩니다.
- 세션이 만료되면 로그인 페이지로 자동 이동됩니다.
